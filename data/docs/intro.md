# Введение в алгоритмы

Алгоритм — это точная последовательность действий, приводящая к решению поставленной задачи.

## Свойства алгоритмов:
1. **Дискретность** — алгоритм должен состоять из отдельных шагов.
2. **Определенность** — каждый шаг должен быть понятен исполнителю.
3. **Конечность** — алгоритм должен завершаться за конечное число шагов.
4. **Массовость** — алгоритм должен решать класс однотипных задач.

### Пример:
```python
def hello_world():
    print("Hello, World!")
```
