# Основы HTML

HTML (HyperText Markup Language) — это язык разметки гипертекста.

## Основные теги:
- `<html>` — корневой элемент
- `<head>` — метаинформация
- `<body>` — основное содержимое
- `<h1>` - `<h6>` — заголовки
- `<p>` — параграф
- `<a>` — ссылка
